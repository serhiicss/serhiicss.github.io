--- layout: post title: "Easily live reload your website with Browsersync (Windows)" meta: Reload website with Browsersync date: 2016-11-11 categories: tools ---

First of all, let's install node.js

  1. Go to https://nodejs.org/
  2. Download LTS (recommended) version
  3. Install and restart computer

Open a command prompt window, type in node -v and hit enter. If you see version number, node.js is installed properly.


Now, let's install and use BrowserSync.

  1. In the command prompt type: npm install -g browser-sync and hit enter.
  2. Navigate to your project folder.
  3. Type in browser-sync start --server --files "*.html, *css/.css"
  4. Change directories according to your project structure.
  5. Hit enter and enjoy!

Update: Make sure you have an index.html file in your project directory.
